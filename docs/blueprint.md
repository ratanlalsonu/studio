# **App Name**: ApnaDairy Online

## Core Features:

- Product Catalog: Display a range of dairy products (milk, ghee, paneer, khoya, chhena, curd) with options for liter and milliliter quantities.
- Shopping Cart: Allow customers to add products to a shopping cart with quantity selection.
- User Authentication: Implement user login and signup functionality.
- Order Placement Popup: Display a confirmation popup with order details before finalizing the purchase.
- Order Tracking: Enable users to track their order status and view order history.
- Payment Options: Offer multiple payment methods, including cash on delivery (COD), UPI, and QR code.
- Order Confirmation Notification: Provide real-time notifications to users upon successful order confirmation.

## Style Guidelines:

- Primary color: Earthy brown (#A67B5B) evoking natural farm products.
- Background color: Off-white (#F5F5DC) for a clean, dairy-like feel.
- Accent color: Light green (#90EE90) to symbolize freshness and nature.
- Body and headline font: 'PT Sans', a humanist sans-serif font providing a modern and friendly look for both titles and body text.
- Use line icons depicting cows, buffaloes, milk drops, and other relevant imagery.
- Employ a grid-based layout for product listing and category pages.
- Use subtle transitions on button hovers and product interactions to add a modern touch.
